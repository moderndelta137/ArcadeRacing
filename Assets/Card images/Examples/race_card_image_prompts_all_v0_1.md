# Race Card Image Prompts — Full 33-Card Set

This file collects the prompts used or reconstructed for the race-card image generations.

Note: the most recent **Muscle Car / Mustang GT500** prompts are preserved nearly verbatim from the generation call. Earlier prompts are reconstructed from the generation instructions and card specifications because the image generation tool does not expose every prior prompt verbatim after the fact. They are written in the same structure and intent that was used: same card layout, same car, same card text, and same illustration direction.

---

## Shared Card Layout Prompt

Use this shared layout instruction for all player race cards:

```text
Create a single poker-card-size player race card. Keep the same established racing card template: top-left circular speed requirement badge, top-center card name, large illustration panel below the title, keyword timing box just above the lower text box on the left, large bottom-half effect text box, and card type at the bottom right. Use the speed requirement icon style where maximum speed is a white circular road sign with red ring and black number, minimum speed is a solid blue circular sign with white number, and no requirement uses a neutral circular ANY badge. Use a full-color 1990s Japanese street-racing manga / Initial D-inspired color-page style, with strong black ink, speed lines, halftone texture, dynamic car action, and readable card UI. Do not use copyrighted logos or characters.
```
